实验三实验报告

实验报告：利用云平台搭建个人博客

实验目的：

熟悉个人博客系统的搭建

实验步骤与内容：

  1. 准备工作：安装两个本次需要用到的软件，node.js和git

    并查看两者版本，确定成功安装。

  2. 注册github账号并创建个人仓库。

  3. 创建blog文件夹，并右键使用git bash打开命令行。

  4. 使用命令cnpm install -g hexo-cli安装hexo

  5. 使用命令hexo init初始化

  6. 使用命令hexo s然后在本地浏览器输入localhost:4000，可以本地浏览博客。

  7. 设置ssh，生成sshkey

  8. 在本地进行验证

  9. 上传测试博客，在文件_config最后进行修改。

  10. 新建一篇文章并进行查看，成功建成

  11. 部署到github上

    结论分析与体会：

    结论分析：

    在这次实验中， 我们成功注册了 Github 账号， 在 github 中创建了仓库， 并 且成功在本地 windows 系统上搭建了 Hexo 环境并实现个人博客连接， 实验成功。

    心得体会：

    本次实验不仅让我了解了 gi thub 平台的理论知识和实际注册运用， 也让我认识 了 git 和 hexo 的强大能力、实用性和前沿技术性，学习了在 github 上进行账号注册和仓库 构建方法， 对 git、 hexo 和 github 之间的关系的认识加深了； 也获得一次宝贵的线上实验 的体验， 对云计算这门学科有了更深厚的兴趣。
